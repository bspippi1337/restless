RESTLESS

universal api client • tui / cli / automation

Bygd for fart, kontroll og ro i kroppen

Gjør API-kaos om til flyt.

Restless er en universal API-klient med proff TUI, raske workflows og automagi som føles som håndverk: små detaljer, presise svar, og en UI som holder seg kald selv når nettet er varmt.

Kom i gang
Request builder Headers, auth, body, retries, templates. Ryddig som et verkstedbord.
Discovery & presets Samler endepunkter, lagrer flows, og bygger “default good choices”.
CLI/TUI først Funker i terminal, CI og på farten. Null stress, bare signal.

Workflow design

Domene inn → struktur ut → trygt videre

1) Skriv inn et domene Restless starter med én ting du faktisk vet: hvor API-et bor.
2) Oppdag docs og endpoints Leser og finner det som finnes, ikke det som “kanskje” finnes.
3) Verifiser forsiktig Trygge metoder, harde budsjetter, ingen brute force, ingen auth-bypass.
4) Du får en seedet request Du starter ikke fra null. Du starter fra et godt utgangspunkt.

Kom i gang

Bygg lokalt eller kjør fra binær

$ git clone https://github.com/bspippi1337/restless
$ cd restless
$ go mod tidy
$ make build
$ ./bin/restless
Tips: I wizard, skriv domene og trykk Ctrl+D for discovery.
