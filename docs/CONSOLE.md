# Restless Console (v1)

The Console is an interactive REPL that helps you build working requests for your active API profile, run them, and save them as reusable snippets.

## Start

```bash
restless console --profile openai
```

Optional base override:

```bash
restless console --profile openai --base https://api.openai.com
```

## Core flow

Inside the console:

```text
suggest
run
save models
snippets
use models
```
