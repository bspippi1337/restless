Why Restless

        

          Most API tools assume you already have a spec, a token, a base URL, and a plan.
          Restless starts earlier: from a domain. It does the heavy lifting.
        


        

          

            
Domain-first

            
Type a domain. Restless hunts for entry points and organizes evidence, so you can move fast without guessing.

          

          

            
Profiles that grow

            
Save what works. Profiles remember base URLs, auth hints, and discovered endpoints. No brittle copy pasta.

          

          

            
Snippets you can trust

            
Generate working request snippets (curl or native). Keep them tidy, searchable, and reusable.

          

          

            
SSE & streaming

            
First-class streaming support, so long-running tasks feel alive instead of frozen in time.

          

          

            
Built-in “Doctor”

            
Automagic sanity checks. Fix what can be fixed, and print the rest with useful next steps.

          

          

            
Terminal UI

            
Even in CLI-mode, Restless can draw a friendly UI and guide you like a co-pilot, not a lecture.

Demo flow

        
Copy, paste, ship. This is the happy path.


        

          

            
Discover and save a profile

            
restless discover openai.com --save-profile openai

            
Find entry points and store a reusable profile.

          


          

            
Launch the interactive console

            
restless console --profile openai

            
Generate queries, run, and save snippets without leaving the session.

          


          

            
List and run snippets later

            
restless snippets list --profile openai
restless snippets run  --profile openai models

            
Over time, your snippet shelf becomes a personal API cookbook.

          


          

            
Stream responses (SSE)

            
restless request --profile openai GET /v1/models --stream

            
No dead silence, no frozen terminals, just live output.

          

        

      

    


    

      

        
Get Restless

        
Start with a source build today. Releases can attach binaries as the CI matures.


        

          

            
Linux / Debian / WSL

            
git clone https://github.com/bspippi1337/restless
cd restless
make build-all
./dist/restless --help

          

          

            
Termux

            
pkg update -y
pkg install -y git golang make
git clone https://github.com/bspippi1337/restless
cd restless
make build-all
./dist/restless --help

          

        


        

          

            
Tip

            
Restless includes a fix-all script in scripts/fixall.sh for Termux/Debian environments.
