Quickstart

Run directly from source (works on Debian/Termux when CGO is off).

git clone https://github.com/bspippi1337/restless
cd restless
go mod tidy
CGO_ENABLED=0 go test ./... -count=1 -tags "netgo osusergo"
CGO_ENABLED=0 go run ./cmd/restless --help

Console (interactive)

Build + run requests interactively, then save reusable snippets.

restless console --profile openai
# inside:
suggest
run
save models
snippets
use models

Docs: docs/CONSOLE.md and docs/SNIPPETS.md

Snippets

Run and export saved calls with a single command. Snippets track success + latency.

restless snippets list --profile openai
restless snippets run  --profile openai models
restless snippets export --profile openai models --format curl

History

Every run appends a JSONL record for analysis and future “smart suggestion” features.

~/.config/restless/history/openai.jsonl
