# restless

Escape goto-10 development.

Restless shows you what exists before you know what you're looking for.

Point it at an API.
Point it at a repo.
Point it at a system you just opened and don't trust yet.

    restless map .

Restless discovers structure automatically:

- endpoints
- schema shape
- topology
- surface changes over time

No config.
No guessing.
No outdated docs required.

---

## Why restless exists

Modern systems are bigger than their documentation.

When you open something unfamiliar, the first problem isn't editing code.

It's orientation.

Restless answers:

> what actually exists here?

before you even know what to ask.

---

## What restless is

Restless is situational awareness for developers.

Use it when:

- onboarding to a repo
- exploring an API
- validating documentation
- auditing service surfaces
- preparing refactors
- comparing environments

Run it early.
Run it often.
Avoid the loop.
